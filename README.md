## Getting Started

To use SCSS(SASS), you need to install SASS

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VegePizza/vegepizza.github.io.git
   ```
2. Install NPM packages (SASS)
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm run sass
   ```
